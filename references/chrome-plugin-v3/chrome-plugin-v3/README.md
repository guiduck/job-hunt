# Extensão Cuponeria v3

## Requisitos mínimos

- VSCode
- Node 16+
- Yarn 1.22.4+
- Extensões para o VSCode:
  - (ESLint, da Microsoft) dbaeumer.vscode-eslint
  - (Prettier, da Prettier) esbenp.prettier-vscode

## Instruções

Este é um projeto feito em React. Para executar, é necessário seguir os passos abaixo.

### Instalação

- Executar `yarn` ou `yarn install`

### Execução

- Executar `yarn dev` ou `yarn run dev`
- Acessar, no navegador, a página [http://localhost:3000](http://localhost:3000). Dê preferencia ao Google Chrome
