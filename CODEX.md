# Codex Entry Point

Use `AGENTS.md` as the main repository instructions for Codex and other non-Cursor agents.

Codex-specific workflow mirrors live in `.codex/skills/`. When a task maps to an existing Cursor
skill, load the matching `.codex/skills/<skill-name>/SKILL.md` file and follow it together with the
shared project context in `README.md`, `docs/`, `.specify/`, and `specs/`.

Cursor remains supported through `.cursor/rules/specify-rules.mdc` and `.cursor/skills/**`.
